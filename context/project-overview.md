# AIHues 项目上下文

## 项目概况
- **名称**: AIHues — AI Vibe Navigator
- **定位**: 出海增长工具导航站 + AI 工具大全
- **风格参考**: https://www.gingiris.tools/
- **配色**: 暖琥珀色 #b45309 为主 + 彩虹色 Logo
- **部署**: GitHub Pages

## 技术栈
- **前端**: 纯 HTML/CSS/JS（无框架）
- **开发**: Live Server
- **版本控制**: Git (GitHub)
- **内容管理**: 飞书文档

## 核心功能
1. 100+ AI 工具导航（出海增长 / AI 产品 / 开源 / 设计 / 开发 / 小工具）
2. 积分系统（credit.js）— 玩游戏赚积分，使用工具消耗积分
3. 小游戏中心（日历幸运签 / 老虎机 / 投篮）
4. AI 智能搜索
5. SEO 文章系统

## 品牌资产
- **Logo**: 小机器人 × 彩虹色环（v4b）
- **Slogan**: "找到你的 AI vibe"
- **Tone**: 简约有趣、实用主义
