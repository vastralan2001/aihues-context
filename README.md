# AIHues Context & Prompts Repository

AIHues 项目的智能助手工作仓库。

## 🔗 关联仓库

| 仓库 | 用途 | 链接 |
|------|------|------|
| **ai-hues-v6** | 代码仓库 | https://github.com/vastralan2001/ai-hues-v6 |
| **aihues-context** (本仓库) | 项目上下文、日报、决策记录 | — |
| **personal-context** | 个人通用记忆、跨项目上下文 | https://github.com/vastralan2001/personal-context |

## 目录结构

```
aihues-context/
├── prompts/        # 每次需求 prompt
├── context/        # 项目背景、技术栈、设计规范
├── decisions/      # 关键决策记录
├── daily/          # 日报（YYYYMMDD.md）
└── weekly/         # 周报（2026-W##.md）
```

## 使用说明

- **prompts/**: 每次给 AI 的 prompt 都存一份，方便回溯
- **context/**: 项目状态、技术栈、设计规范等背景信息
- **decisions/**: 重要决策的记录（为什么选这个方案）
- **daily/**: 每日工作摘要，自动归档
- **weekly/**: 每周复盘，总结进展和教训
